# microsstv

# Install
`$ yarn`
# Run
## Development
`$ yarn run dev`
## Staging
`$ node index.js --staging --config=/absolute/path/to/config/dir`
## Production
`$ node index.js --production --config=/absolute/path/to/config/dir`
